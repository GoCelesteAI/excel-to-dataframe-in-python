## What's in here

Three pandas patterns for reading an `.xlsx` workbook into a DataFrame.

- `excel_to_dataframe.py` — the demo script
- `stock_summary.xlsx` — sample two-sheet workbook (Prices: 420×8, Tickers: 14×2)

## Run it

```bash
pip install -r requirements.txt
python excel_to_dataframe.py
```

Output:

```
single sheet:         (420, 8)
all sheets:           [('Prices', (420, 8)), ('Tickers', (14, 2))]
ExcelFile parsed:     prices=(420, 8), tickers=(14, 2)
```

## What it shows

1. `pd.read_excel(FILE, sheet_name="Prices")` — single sheet by name.
2. `pd.read_excel(FILE, sheet_name=None)` — all sheets at once as a dict keyed by sheet name.
3. `with pd.ExcelFile(FILE) as xl: xl.parse("Prices")` — context manager that opens the workbook once for repeated reads.

All three use the `openpyxl` engine that ships with pandas. No extra install needed for read-only use cases.
